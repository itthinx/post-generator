A sample post generator for WordPress, supports custom post types.

== Description ==

This plugin is intended to be used as a sample post generator for WordPress.

It's purpose is to provide an automated way of creating even very large sets
of posts, useful in providing a test environment for performance benchmarks
and use case testing.

To create sample products for WooCommerce, it's recommended to rather use [WooCommerce Product Generator](http://wordpress.org/plugins/woocommerce-product-generator/).

== Installation ==

= Dashboard =

Log in as an administrator and go to <strong>Plugins > Add New</strong>.

Type <em>Post Generator</em> in the search field and click <em>Search Plugins</em>, locate the <em>Post Generator<em> plugin by <em>itthinx</em> and install it by clicking <em>Install Now</em>.
Now <em>activate</em> the plugin to be able to generate sample products.

= FTP =

You can install the plugin via FTP, see [Manual Plugin Installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
